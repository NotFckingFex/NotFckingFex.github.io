Projekte:

WebScraping mit Daten des Robert-Koch-Instituts.

Entwickelt in Python mit den Programmbibliotheken Beautiful Soup 4, Pandas und Selenium.

Es geht darum, die Daten des RKI's über ein Programm vollautomatisch alle 24h auszulesen, diese Daten werden in einer Datenbank gespeichert und überschrieben.

Daraufhin werden die Daten aus der Datenbank, mithilfe von der Programmiersprache PHP, auf einer Webseite angezeigt werden. - der Code wird öffentlich auf meinem Github sein.
